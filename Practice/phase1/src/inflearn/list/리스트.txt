리스트
 - 중복을 허용하며 순서가 있는 연속된 데이터를 표현하는 자료구조
 - 배열과 달리 사용하는 중에 크기가 자동으로 조절된다.

 자바가 제공하는 대표적인 리스트
 - Vector : 모든 오퍼레이션에 한 순간에 한 쓰레드만 접근 가능
 - ArrayList : 여러 쓰레드가 접근 가능함(동기화에 필요한 작업을 할 필요가 없음)
 - LinkedList

 ArrayList
 - 추가 및 삭제 O(1) 하지만 최악의 경우에는 O(N) , 중간에 넣을땐 O(N), contains같을땐 O(N)
 - 조회 O(1)

 LinkedList : 순서만 있고 개념적인 인덱스만 있는거지 구현되어있는 index는 없어요
 - 추가 및 삭제 O(1) ,, 실질적으로는 O(N)
 - 조회 O(N)

 문제 1. LinkedList 뒤집기
 단일 연결 리스트를 뒤집는 함수를 구현하라
 예) 1 -> 2 -> 3 => 3 -> 2 -> 1

 문제 2. 끝에서 n번째 노드 찾기
 단일 연결 리스트의 끝에서 n번째 위치한 노드를 찾는 함수를 구현하라.
 예) findFormLast(1 -> 4 -> 2 -> 3,2), 끝에서 2번째에 위치한 2를 리턴한다.

 문제 3. 정렬된 연결 리스트에서 중복 노드 제거하기
 정렬된 연결 리스트에서 중복가 노드를 제거하는 함수를 구현하라
 예) 1 -> 1 -> 1 -> 2 -> 3 -> 3 => 1 -> 2 -> 3

 문제 4. 원형 연결 리스트
 주어진 연결리스트가 원형 연결 리스트인지 단일 연결 리스트인지 확인하는 함수를 구현하라.
 예 1 -> 2 -> 3 -> 1 => true
 예 1 -> 2 -> 3 -> 2 => true
 예 1 -> 2 -> 3      => false
